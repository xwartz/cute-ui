# cute-ui
Cute is a UI component framework.

## 开发环境配置

1. 安装 `Nodejs 4.0+`

2. `npm install`

3. 安装 `gulp`, 因为我一直用的全局的

4. `gulp watch`

5. 'pm2 start app --watch'

## 目录说明

1. `views` 下是各个组件的 `handlebars` 模板

2. `public/scss` 是 scss 源码

3. `app.js` 是 `express` 启动文件
